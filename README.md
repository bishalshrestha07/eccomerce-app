# MeroDokan
## Online Platform to buy and sell products online

### Technology used:
- Backend: Django, DRF
- Frontend: React Js
- Database: Mysql


## Features

- Let user to add, remove products and details
- JWT Authentication
- Paypal payment Integration
- Product Tracking
